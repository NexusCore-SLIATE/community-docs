# 🎓 Freshers Workspace Setup Guide

Welcome to NexusCore! This guide will take you from a freshly cloned workspace to submitting your very first production-grade pull request.

## 🛠️ Required Baseline Tooling
Before writing any code, download and configure these core tools:
1. **Operating System:** Linux Mint is highly recommended for our software and security tracks.
2. **Terminal Emulator:** Ensure `git` is installed (`sudo apt install git`).
3. **IDE:** VS Code or VSCodium configured with standard linting extensions.

## 🔑 Git Initialization
Set up your global configuration using your official GitHub handle details:
```bash
git config --global user.name "Your Name"
git config --global user.email "your-github-email@example.com"
